function setCookie
Send a coookie to the client browser.
Declaration:
     function setCookie( string cookie )
Description:
It is important that you form the cookie correctly and escape the values.

A cookie takes the form:

key=value; [expires=DATE;] [path=PATH;]

Where expires and path are optional. The main part is the key=value. It is important that the expires date is given in RFC-822 format. This format looks like "Sat, 14 Sep 2002 21:57:39 +0000" and is the format given by UNIX command date -R.

If you wanted to, it is possible to combine this with the serialize module and String.base64encode to store user objects in cookies rather than clog up a database. Although this could be considered a \ secuirity risk.

Parameters:
    Parameter #1: string cookie - The cookie to get
