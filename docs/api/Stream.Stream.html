function ungetc
Push a character back onto the internal buffer
Declaration:
    function ungetc( string c )
Description:
This function pushes a character back into the internal buffer. This character can be obtained by calling getc() or read(). For instance if the buffer contained 'erite\n' and .ungetc('f') was called, the internal buffer would now be 'ferite\n'. A readln() would result in 'ferite\n' being returned.
Parameters:
