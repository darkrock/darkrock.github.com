class Transform

Provided By: xslt.fec

The core class of the XSLT module, this class provides the mechanism for doing XSLT transformations.

Description

Objects from this class are instantiated with an xsl stylesheet and can be applied to more than one XML.TreeParser object.

Example

uses "xml";
uses "xslt";
uses "console";

object stylesheet = new XSLT.Transform("test.xsl");
object parser = new XML.TreeParser();
parser.parseFile("test.xml");

stylesheet.apply( parser );
Console.println( "======== xslt toString() =======" );
Console.println( stylesheet.toString() );

parser = stylesheet.getXMLTreeParser();
Console.println( "-------- parser toString() -----" );
Console.println( parser.toString() );
